Auf dem raspberry muss java 8 installiert sein.

die befehle müssen im diesem verzeichniss ausgeführt werden.
der content ordner muss noch das root verzeichniss von einem lokalen webserver sein.

dann den server starten:
java -jar cf-helloworld-server-1.1.0-SNAPSHOT.jar

und den client ausführen
java -jar cf-helloworld-client-1.1.0-SNAPSHOT.jar

und die ausgabe vom client zuschicken.
